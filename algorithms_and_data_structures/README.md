### 3. Max-–∫—É—á–∞ –Ω–∞ –æ—Å–Ω–æ–≤–µ –º–∞—Å—Å–∏–≤–∞

### 13. –¢–æ–ø–æ–ª–æ–≥–∏—á–µ—Å–∫–∞—è —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –∏ —Å–≤—è–∑–Ω–æ—Å—Ç—å –≤ –æ—Ä–≥—Ä–∞—Ñ–∞—Ö  

**–ó–∞–¥–∞—á–∞**  
–†–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å –∞–ª–≥–æ—Ä–∏—Ç–º—ã –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å –Ω–∞–ø—Ä–∞–≤–ª–µ–Ω–Ω—ã–º–∏ –≥—Ä–∞—Ñ–∞–º–∏: —Ç–æ–ø–æ–ª–æ–≥–∏—á–µ—Å–∫—É—é —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫—É, –ø—Ä–æ–≤–µ—Ä–∫—É —Ü–∏–∫–ª–æ–≤ –∏ –Ω–∞—Ö–æ–∂–¥–µ–Ω–∏–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç —Å–∏–ª—å–Ω–æ–π —Å–≤—è–∑–Ω–æ—Å—Ç–∏. –¢–æ–ø–æ–ª–æ–≥–∏—á–µ—Å–∫–∞—è —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π –ª–∏–Ω–µ–π–Ω–æ–µ —É–ø–æ—Ä—è–¥–æ—á–∏–≤–∞–Ω–∏–µ –≤–µ—Ä—à–∏–Ω –∞—Ü–∏–∫–ª–∏—á–Ω–æ–≥–æ –Ω–∞–ø—Ä–∞–≤–ª–µ–Ω–Ω–æ–≥–æ –≥—Ä–∞—Ñ–∞ (DAG), –ø—Ä–∏ –∫–æ—Ç–æ—Ä–æ–º –¥–ª—è –∫–∞–∂–¥–æ–≥–æ —Ä–µ–±—Ä–∞ $(u,\ v)$ –≤–µ—Ä—à–∏–Ω–∞ $u$ –ø—Ä–µ–¥—à–µ—Å—Ç–≤—É–µ—Ç –≤–µ—Ä—à–∏–Ω–µ $v$, –∏ –º–æ–∂–µ—Ç –±—ã—Ç—å —Ä–µ–∞–ª–∏–∑–æ–≤–∞–Ω–∞ —Å –ø–æ–º–æ—â—å—é –ø–æ–∏—Å–∫–∞ –≤ –≥–ª—É–±–∏–Ω—É (DFS) –∏–ª–∏ –∞–ª–≥–æ—Ä–∏—Ç–º–∞ –ö–∞–Ω–∞ (Kahn's Algorithm), –æ–±–∞ –∏–∑ –∫–æ—Ç–æ—Ä—ã—Ö —Ä–∞–±–æ—Ç–∞—é—Ç –∑–∞ $O(V + E)$. –ü—Ä–æ–≤–µ—Ä–∫–∞ –Ω–∞–ª–∏—á–∏—è —Ü–∏–∫–ª–æ–≤ –Ω–µ–æ–±—Ö–æ–¥–∏–º–∞, —Ç–∞–∫ –∫–∞–∫ —Ç–æ–ø–æ–ª–æ–≥–∏—á–µ—Å–∫–∞—è —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –Ω–µ–≤–æ–∑–º–æ–∂–Ω–∞ –ø—Ä–∏ –∏—Ö –Ω–∞–ª–∏—á–∏–∏, –∏ –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º DFS –∏ –æ—Ç–º–µ—Ç–æ–∫ —Å–æ—Å—Ç–æ—è–Ω–∏–π –≤–µ—Ä—à–∏–Ω (–Ω–µ –ø–æ—Å–µ—â–µ–Ω–∞, –≤ –æ–±—Ä–∞–±–æ—Ç–∫–µ, –æ–±—Ä–∞–±–æ—Ç–∞–Ω–∞), —Ç–∞–∫–∂–µ –∑–∞ $O(V + E)$. –ö–æ–º–ø–æ–Ω–µ–Ω—Ç—ã —Å–∏–ª—å–Ω–æ–π —Å–≤—è–∑–Ω–æ—Å—Ç–∏ ‚Äî —ç—Ç–æ –ø–æ–¥–≥—Ä–∞—Ñ—ã, –≤ –∫–æ—Ç–æ—Ä—ã—Ö –ª—é–±—ã–µ –¥–≤–µ –≤–µ—Ä—à–∏–Ω—ã –¥–æ—Å—Ç–∏–∂–∏–º—ã –¥—Ä—É–≥ –∏–∑ –¥—Ä—É–≥–∞, –∏ –º–æ–≥—É—Ç –±—ã—Ç—å –Ω–∞–π–¥–µ–Ω—ã –∞–ª–≥–æ—Ä–∏—Ç–º–æ–º –ö–æ—Å–∞—Ä–∞–π—é –∏–ª–∏ –¢–∞—Ä—å—è–Ω–∞, –∏—Å–ø–æ–ª—å–∑—É—é—â–∏–º–∏ –¥–≤–∞ –æ–±—Ö–æ–¥–∞ –≥—Ä–∞—Ñ–∞ –∏ —Ä–∞–±–æ—Ç–∞—é—â–∏–º–∏ –∑–∞ $O(V + E)$. –≠—Ç–∏ –∞–ª–≥–æ—Ä–∏—Ç–º—ã —à–∏—Ä–æ–∫–æ –ø—Ä–∏–º–µ–Ω—è—é—Ç—Å—è –≤ –∞–Ω–∞–ª–∏–∑–µ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–µ–π, –∫–æ–º–ø–∏–ª—è—Ü–∏–∏ –ø—Ä–æ–≥—Ä–∞–º–º –∏ –∞–Ω–∞–ª–∏–∑–µ —Å–µ—Ç–µ–π.  


### 14. –ö–ª–∞—Å—Ç–µ—Ä–∏–∑–∞—Ü–∏—è –∏ DSU

**–ó–∞–¥–∞—á–∞**  
–†–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å –∞–ª–≥–æ—Ä–∏—Ç–º—ã, –∏—Å–ø–æ–ª—å–∑—É—é—â–∏–µ —Å—Ç—Ä—É–∫—Ç—É—Ä—É DSU (Disjoint Set Union), —Ç–∞–∫–∂–µ –∏–∑–≤–µ—Å—Ç–Ω—É—é –∫–∞–∫ —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –Ω–µ–ø–µ—Ä–µ—Å–µ–∫–∞—é—â–∏—Ö—Å—è –º–Ω–æ–∂–µ—Å—Ç–≤ (Union-Find), –¥–ª—è –¥–≤—É—Ö —Ä–∞–∑–ª–∏—á–Ω—ã—Ö –∑–∞–¥–∞—á: –∫–ª–∞—Å—Ç–µ—Ä–∏–∑–∞—Ü–∏–∏ –∏ —Ä–µ—à–µ–Ω–∏—è –•–∞–Ω–æ–π—Å–∫–∏—Ö –±–∞—à–µ–Ω. ### 15. –û–±—Ä–∞–±–æ—Ç–∫–∞ JavaScript-—Ñ–∞–π–ª–æ–≤
